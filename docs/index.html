For-Schleife in Java

Einführung

Die for-Schleife in Java ist eine Kontrollstruktur, die es ermöglicht, einen Codeblock mehrfach auszuführen. Sie ist besonders nützlich, wenn die Anzahl der Durchläufe im Voraus bekannt ist.

In dieser Präsentation lernst du die Grundlagen der for-Schleife und siehst praktische Beispiele.

Grundsyntax

Die Syntax der for-Schleife in Java: 

for (Initialisierung; Bedingung; Aktualisierung) {
    // Code, der wiederholt ausgeführt wird
}
  • Initialisierung: Wird einmal zu Beginn ausgeführt (z. B. Zählvariable initialisieren).
  • Bedingung: Wird vor jedem Durchlauf geprüft. Wenn false, wird die Schleife beendet.
  • Aktualisierung: Wird nach jedem Durchlauf ausgeführt (z. B. Zählvariable erhöhen).

Beispiel 1: Zahlen ausgeben

Dieses Beispiel gibt die Zahlen von 1 bis 5 aus: 

for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

Ausgabe: 

1
2
3
4
5

Beispiel 2: Arrays durchlaufen

Mit einer for-Schleife kannst du einfach durch Arrays iterieren: 

String[] namen = {"Anna", "Ben", "Clara", "David"};

for (int i = 0; i < namen.length; i++) {
    System.out.println(namen[i]);
}

Ausgabe: 

Anna
Ben
Clara
David

Beispiel 3: Verschachtelte Schleifen

Du kannst for-Schleifen auch verschachteln, z. B. für eine Multiplikationstabelle: 

for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 5; j++) {
        System.out.print(i * j + " ");
    }
    System.out.println();
}

Ausgabe: 

1 2 3 4 5
2 4 6 8 10
3 6 9 12 15
4 8 12 16 20
5 10 15 20 25

Zusammenfassung

Die for-Schleife ist eine mächtige Kontrollstruktur in Java, die dir hilft, Codeblöcke effizient zu wiederholen. Sie besteht aus Initialisierung, Bedingung und Aktualisierung.

Typische Anwendungsfälle sind:

  • Zahlenbereiche durchlaufen
  • Arrays und Listen verarbeiten
  • Verschachtelte Schleifen für komplexe Aufgaben

Übe die Beispiele und experimentiere mit eigenen Schleifen!
